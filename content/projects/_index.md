- **LazyLang** ([GitHub](https://github.com/Lumberj3ck/LazyLang)) - Terminal-based language learning app that lets you converse with an AI and translate unfamiliar words inline without leaving the terminal.

- **LemoSync** ([GitHub](https://github.com/Lumberj3ck/LemoSync)) - Peer-to-peer chat app with end-to-end encryption (ECDH + AES-GCM) and offline message queueing.

- **pass_web** ([GitHub](https://github.com/Lumberj3ck/pass_web)) - Web UI for the Unix `pass` password manager with client-side OpenPGP encryption/decryption and PGP challenge-response auth.

- **GroundHog** ([GitHub](https://github.com/Lumberj3ck/GroundHog)) - AI day-planning assistant that turns notes/journal entries into Google Calendar events and tracks daily progress.

- **valentiny** ([GitHub](https://github.com/Lumberj3ck/valentiny)) - Customizable Valentine postcard web app built with Vue.

- **valentiny_api** ([GitHub](https://github.com/Lumberj3ck/valentiny_api)) - FastAPI backend for a customizable Valentine postcard website.

- **Economic_cycles_analyser** ([GitHub](https://github.com/Lumberj3ck/Economic_cycles_analyser)) - German economic forecasting project with data collection, LSTM training, XAI analysis, and an XGBoost cycle classifier exposed via Streamlit.

- **it_tools** ([GitHub](https://github.com/Lumberj3ck/it_tools)) - Vite-based web app of small IT utilities with unit tests (Vitest) and end-to-end tests (Playwright).

- **file_browser** ([GitHub](https://github.com/Lumberj3ck/file_browser)) - Terminal file browser and process manager built with Textual (bookmarks, fuzzy search, and process killing).

- **ecomerse_website** ([GitHub](https://github.com/Lumberj3ck/ecomerse_website)) - Full-stack e-commerce app built with Vue, Node/Express, and SQLite including auth, cart, and PayPal payments.
